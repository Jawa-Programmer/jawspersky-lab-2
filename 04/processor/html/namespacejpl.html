◆ OPERATION_TYPE

enum jpl::OPERATION_TYPE

тип операции. используется для внутреннего пользования. При создании своего типа операции, следует использовать OP_EXTER.

Элементы перечислений
OP_VAR 

операция статического выделения. Обрабатывается обособленно, на этапе компиляции

OP_EXTER 

тип обычной операции.

OP_NONE 

тип обстрактной операции, генерируемый command конструктором по умолчанию. Не должен быть вызван для исполнея.
